== Description ==

The Lobby Chawing provides 1 interface for many chatboxes, Read Only channels(Broadcast Mode), and Live Help (private messaging with admin, departments or specific users) to all be combined together.
The Lobby enables users to easily access and communicate with the relevant information in an intuitive real time interface. The Lobby feature can be embedded into a website via HTML iFrame code or "Float" at the bottom of your site via Javascript code provide. As with all Chatwing features the Lobby can be completely customized and designed to fit your communication needs. The Lobby also syncs with a custom iOS+Android app Chatwing can build and power for you.

== Installation ==

1. Upload content of the plugin zip file to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to chatwing.com, log in to your account and get the access token for WordPress plugin in order to use the plugin - [detail guide here](https://youtu.be/rHDl8Z_tgGE).

== Frequently asked questions ==
